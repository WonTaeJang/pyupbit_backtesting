# pyupbit를 이용한 백테스팅 페이지 

## 1. 개요
pyupbit API를 활용한 백테스팅 페이지 (app.py)
커맨드 예: python pyupbit_backtesting/server/app.py

## 2. 개발 명세
### 2.1 프로그램 사양
- Python 3.8.9
- pyupbit_backtesting file (132KB)
- window os

## 3. 상세내용
- 변동성 돌파전략을 이용한 백테스팅
- 투자금액 설정후 종목, 봉단위, 범위, K 값을 설정후 결과값을 확인하는 프로그램
